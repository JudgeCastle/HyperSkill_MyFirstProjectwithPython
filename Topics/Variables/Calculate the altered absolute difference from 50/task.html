Calculate the altered absolute difference from 50

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given a number, your task is to find the absolute difference between a number and 50. If the given number is above 50, then the output should be three times the absolute difference. Otherwise, the output should be the absolute difference itself.
Sample Input:
25

Sample Output:
25


Sample Input:
65

45
